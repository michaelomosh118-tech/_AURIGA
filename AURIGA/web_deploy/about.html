About Auriga

Auriga is a vision-assistance ecosystem from DrakoSanctis — an Android app and progressive web app that pair on-device computer vision with audio-first navigation. It is built and maintained by a one-person team and ships as a single, dependency-light bundle so it keeps working in places the cloud doesn't reach.

Project mission

See beyond sight. Auriga turns a phone's camera, sensors and speech into a calm spatial guide — for low-vision users, for people navigating unfamiliar rooms, and for anyone who wants the world narrated rather than displayed.

How submissions are handled

Every submission is delivered to the project Gmail and mirrored to the project's GitHub Issues tracker as the long-term ledger. If you provide a reply-to email, an automatic acknowledgement with your ticket ID is sent to you within seconds — quote that ID in any future correspondence and we can pick the thread up instantly.

Bug and "Need help" submissions require a reply-to email so we can actually reach you with a fix. Ideas and general feedback don't.
